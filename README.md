# How to run:
## Run it locally
install the dependancies:  
`pip install -r requirement.txt`  
run the bot:  
`python main.py`  
## Or use docker
If you haven't built the image:  
`docker-compose up --build -d`  
If you've already built the image:  
`docker-compose up -d`
